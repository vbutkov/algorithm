Есть N полосок.
Полоски лежат на одной прямой, у каждой есть координата начала и конца - start[i] и end[i].
Нужно выбрать максимальное кол-во полосок так, чтобы они выбранные полоски
не пересекались между собой.

Пример
N=4
start = {1,1,3,2}
end =   {5,2,4,3}

Ответ:
Кол-во полосок 3
полоски: {start=1, end=2}, {start=2, end=3}, {start=3, end=4}
